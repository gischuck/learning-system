# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-04-02

### Added

#### 核心功能
- ✨ 孩子看板：习惯打卡、星星等级显示、心愿商城
- ✨ 家长后台：规划管理、待办事项、课程管理
- ✨ 星星奖励系统：自动计算星星余额
- ✨ 心愿审批流程：孩子提交 → 家长审批

#### 习惯管理
- 🎯 正面习惯：早睡、早起、收书包
- 🎯 负面习惯：玩游戏、看电视、漏作业
- 🎯 自定义星星奖励/惩罚规则

#### 作业管理
- 📝 布置作业，设置截止日期
- 📝 完成作业奖励星星
- 📝 作业状态跟踪

#### AI 助手
- 🤖 支持阿里云百炼 AI
- 🤖 智能回答 KET/PET、GESP 问题
- 🤖 自定义 prompt 配置

#### 数据同步
- 🔄 Google Sheets 同步
- 🔄 Google Docs 笔记
- 🔄 本地数据库备份
- 🔄 数据导出/导入（JSON）

#### UI/UX
- 🎨 明亮/暗黑主题切换
- 🎨 响应式设计
- 🎨 可爱的等级系统（星光萌芽、星光初现...）

### Technical

- Frontend: React 18 + Vite + Tailwind CSS
- Backend: Node.js + Express + Sequelize + SQLite
- Optional: Google Sheets API, 阿里云百炼 AI
